# Catálogo de Livros - Frontend

Este é o frontend do projeto Catálogo de Livros, desenvolvido com **React**. Permite aos usuários visualizar e alugar livros e fornece um painel de administração para gerenciar o catálogo de livros.

## Funcionalidades

- **Fluxo de Usuário**: Visualização e aluguel de livros.
- **Fluxo de Administrador**: CRUD de livros.
- **Responsividade**: Interface adaptada para dispositivos móveis.
- **Internacionalização**: Suporte a múltiplos idiomas.

## Tecnologias Utilizadas

- React
- Axios para integração com a API
- i18next para internacionalização
- Styled Components para estilo
- React Testing Library e Jest para testes

## Como Executar

1. Clone o repositório:

   git clone https://github.com/lucasOliveira91/catalogo-livros-frontend.git
   cd catalogo-livros-frontend

2. Instale as dependências:

   npm install

3. Execute o projeto:

   npm start

4. Acesse o website em `http://localhost:3000`.

## Testes

Para executar os testes:

npm test

---

## Estrutura do Projeto

- `src/components`: Componentes reutilizáveis.
- `src/pages`: Páginas principais (UserCatalog e AdminPanel).
- `src/services`: Configuração e integração da API.
- `src/i18n`: Configurações de internacionalização.
